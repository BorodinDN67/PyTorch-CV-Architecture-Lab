# PyTorch CV Architecture Lab

Учебный проект на PyTorch для экспериментов с моделями классификации изображений на MNIST.

Главная цель проекта — не получить максимальное качество на MNIST, а руками собрать небольшой ML-пайплайн и разобраться, как связаны между собой основные части проекта: загрузка данных, описание моделей, цикл обучения, сохранение весов, инференс, метрики и логирование экспериментов.

## Цели проекта

- Собрать понятную структуру PyTorch-проекта для экспериментов с классификацией изображений.
- Реализовать и сравнить несколько архитектур нейронных сетей.
- Разобраться с полным циклом обучения и оценки модели.
- Попрактиковаться в работе с метриками, чекпоинтами и логами экспериментов.
- Начать разбираться в transformer-подходах и self-attention на уровне реализации.

## Что реализовано

- Кастомный `Dataset` и `DataLoader` для ч/б изображений в формате MNIST.
- Базовый цикл обучения на PyTorch.
- Сохранение и загрузка весов модели.
- Оценка качества модели.
- Метрики: accuracy и accuracy по классам.
- Полносвязная модель как простой baseline.
- Сверточная нейронная сеть как основной стабильный baseline.
- Экспериментальная реализация ViT / self-attention модели.

## Модели

### Полносвязная сеть

Простая полносвязная нейронная сеть, которая используется как первый baseline.

Она нужна, чтобы проверить работу пайплайна обучения и получить начальную точку сравнения для более сложных моделей.

### CNN

Небольшая сверточная нейронная сеть с двумя сверточными блоками и полносвязными слоями.

Эта модель используется как основной стабильный baseline для задачи классификации изображений.

### ViT / Self-Attention модель

В проекте также есть экспериментальная реализация transformer-подобной модели для изображений.

Цель этой части проекта — не просто использовать готовый transformer из библиотеки, а понять его основные компоненты через самостоятельную реализацию через блоки внимания и работу с размерностями тензоров.

На текущем этапе модель находится в разработке: она обучается и показывает снижение loss, но пока уступает CNN.

## Текущие результаты

| Модель | Accuracy | Статус |
|---|---:|---|
| Полносвязная сеть | ~0.84 | стабильный baseline |
| CNN | ~0.94 | стабильный baseline |
| ViT / Self-Attention | ~0.4–0.5 после ранних запусков | экспериментальная модель |

Результаты получены в ранних экспериментах и могут измениться после доработки проекта.
